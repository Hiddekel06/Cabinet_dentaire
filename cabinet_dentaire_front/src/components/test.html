AU DEBUT DU PROJET git clone "url du projet" on fait ca une fois pour récupérer le projet sur notre machine git pull origin main pour récupérer les dernières modifications du projet avant de commencer à travailler dessus Tu peux commencer à coder Apres avoir fait des modifications sur le projet, on utilise les commandes suivantes pour les enregistrer et les envoyer sur le dépôt distant : git add .
